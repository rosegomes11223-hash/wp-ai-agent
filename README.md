# WP AI Agent 🤖

An autonomous AI agent that writes, translates, and optimizes WordPress content — built for the global web.

## What This Does

- Writes blog posts automatically using Claude AI
- Translates content into Bengali, Arabic, Spanish, French, and more
- Analyzes SEO and suggests improvements
- Connects directly to WordPress via REST API
- Dashboard to manage everything in one place

## Why I Built This

Over 30 crore people speak Bengali worldwide — but there are almost no AI tools that create quality Bengali content for WordPress. This project fills that gap.

I previously built a multilingual legal assistance app (6 languages, 6 countries) for the USAII Global AI Hackathon 2026. This project applies that same multilingual AI experience to WordPress content management.

## Tech Stack

- TypeScript
- Next.js
- Claude AI (Anthropic)
- WordPress REST API
